# 🧠 TechBrain

ServiceNow Closing Notes manager for IT support teams.

## Features
- Save and organize troubleshooting snippets by category
- AI-powered text refinement and grammar correction
- Attach screenshots as visual references
- Add reference links to each snippet
- Copy formatted notes directly to clipboard
- Export/import your snippet library as JSON

## Usage
Visit the live app: https://minawmikhail-ua07.github.io/tech-brain

## Notes
- All data is saved locally in your browser (localStorage)
- Use Export to back up your snippets
- AI Refine feature requires an Anthropic API key
